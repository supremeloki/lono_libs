Welcome to lono_libs's documentation!
=====================================

`lono_libs` is a unified Python library designed to streamline machine learning model evaluation.
It provides a `UnifiedRunner` for consistent model assessment across various classification
and regression metrics, offering automated preprocessing, reporting, and visualizations.
The library aims to empower ML engineers and data scientists to efficiently identify
the best-performing models aligned with their project goals.
